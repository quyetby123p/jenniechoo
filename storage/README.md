# Storage Rules

Khong tao file thu cong trong thu muc nay neu co the.
Hay dung `../scripts/new-artifact.ps1` de tao file voi ten duy nhat, tranh ghi de.

## Jobs

Bot luu ket qua vao:

- `jobs/pending/`: nhap cho duyet.
- `jobs/published/`: da publish.
- `jobs/cancelled/`: da huy.
- `jobs/failed/`: tao/publish that bai.
